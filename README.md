# Sitio web de empresa de mosaicos

Sitio web desarrollado para la presentación de un emprendimiento de mosaicos decorativos, incluyendo productos, proceso de producción y galería visual.

## Funcionalidades

* Página de inicio con presentación del emprendimiento
* Sección de productos
* Descripción del proceso de producción
* Galería de imágenes
* Sección de contacto

## Tecnologías utilizadas

* HTML
* CSS
* JavaScript

## Descripción

Proyecto enfocado en el desarrollo de una página web tipo empresarial, con el objetivo de presentar información de manera clara, estructurada y visualmente atractiva.

## Notas

Este proyecto se centra en la estructura y presentación visual del contenido. Puede ampliarse con funcionalidades como integración con backend, manejo de formularios o contenido dinámico.
